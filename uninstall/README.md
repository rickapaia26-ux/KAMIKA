# KAMIKA - Desinstalar

## Como Desinstalar

1. Abre a pasta do KAMIKA (ou executa `%USERPROFILE%\KAMIKA\uninstall\desinstalar.bat`)
2. Executa `desinstalar.bat`
3. Confirma com **S**
4. Opcional: apagar pasta de downloads e yt-dlp

## O que é removido

- Pasta do programa: `%USERPROFILE%\KAMIKA\`
- Atalho do Desktop: `KAMIKA START.lnk`
- Pasta de downloads: `Desktop\KAMIKA Downloads\` (opcional)
- yt-dlp (opcional)

## Nota

Os downloads existentes nao sao apagados a menos que escolhas essa opcao.
